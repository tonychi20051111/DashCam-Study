# DashCam-Study

主要的行車紀錄器討論區
- Mobile01  -有行車紀錄器的專門討論區

行車紀錄器 功能
- 前後鏡
- G sensor
- ADAS - 車道偏移， 前車啟步
- 語音控制
- 廣角
- GPS 超速
- GPS 軌跡紀錄
- 胎壓TPMS

行車記錄器 價位
 1000 -10000

行車紀錄器 推薦

用戶在意
- 大小 是否會阻擋 
- 安裝位置與方式
- 清晰度 (包含夜間) 拍車牌
- 角度
- 保固
- 價格
- 照相機預警
- 調日期時間
